# Pre-commit Hook Quick Reference

## First-time setup

```bash
./scripts/setup-hooks.sh
```

## Common commands

| Task | Command |
|------|---------|
| **Run hooks on commit** | (automatic) |
| **Run all hooks manually** | `pre-commit run --all-files` |
| **Check specific hook** | `pre-commit run cargo-fmt --all-files` |
| **Update hook versions** | `pre-commit autoupdate` |
| **Skip hooks (rare)** | `git commit --no-verify` |
| **Uninstall hooks** | `pre-commit uninstall` |

## What gets checked

✓ **Code quality**: `cargo fmt`, `cargo clippy`, `cargo test`  
✓ **Contract stability**: SemVer checks on `cdm` and `fed-protocol`  
✓ **Load-bearing invariants**: No `Co-Authored-By:` trailers, English-only Plans.md  
✓ **File quality**: Trailing whitespace, YAML/TOML syntax, no secrets  

## Troubleshooting

**Hook fails?** Fix the issue and re-stage:
```bash
cargo fmt          # Fix formatting
git add -A
git commit -m "..."
```

**First `cargo test` is slow?** It's cached; subsequent runs are faster.

**Want to skip one hook?** (not recommended, but possible)
```bash
# Skip cargo-test only
pre-commit run --all-files -k 'not cargo-test'
```
